## Knowledge Graph — Card-Embedded D3

**Only for analyzed projects** (Personal + Professional). Paid/FYP cards have no graph.

### Behavior
- Card has a **"View Graph"** button
- Click → card expands vertically, D3 force graph renders inside the card
- Graph starts with **root node only** (depth 0)
- Click root → depth 1 appears (folders/top-level files)
- Click any folder node → depth 2 appears (files inside)
- Click any file node → depth 3 appears (symbols)
- Click expanded node again → collapses its children
- Hover any node → tooltip shows AI-generated summary
- Drag nodes, scroll to zoom — fully interactive
- **"Close Graph"** button collapses card back to card height

### Node colors
- Depth 0 (project root): purple `#a855f7`
- Depth 1 (folders): gold `#fbbf24`
- Depth 2 (files): emerald `#10b981`
- Depth 3 (symbols): purple dim `#7c3aed`

    "description": "Full-stack car rental platform with booking, fleet management, and payment integration.",
    "tech": ["Node.js", "React", "PostgreSQL"],

Fields: `slug`, `project`, `category` (`paid`|`fyp`|`professional`), `description`, `tech[]`, `repo`, `role`.
